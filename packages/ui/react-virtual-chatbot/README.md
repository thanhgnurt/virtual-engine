# react-virtual-chatbot

React components and hooks for the high-performance dynamic-height virtual-chatbot.

## Features
- `<ReactVirtualChatbot />` component for virtualized chat lists.
- Integrated `ResizeObserver` for automatic DOM height measurement.
- Built-in "stick-to-bottom" and scroll anchoring logic.

## Installation
```bash
npm install react-virtual-chatbot
```
